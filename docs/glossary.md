# Glossary

| Term | Definition |
|------|------------|
| Dashboard    | The page shown after logging in. It contains all the boards the user has. |
| Power-Ups    | Trello's feature that allow you to connect Trello to other apps like Slack. |
| Template     | A pre-built Trello board equipped with the lists and cards that act as the base for your own specialized trello board. |
| Trello Board | A tool used as a digital whiteboard to organize projects, tasks, or workflows with lists and cards. |
| Trello Card  | A representation of a task or item, taking the form of a block in a list. |
| Trello List  | A container that can hold a column of cards. Trello lists are usually named after a category that represents the cards within it. |
